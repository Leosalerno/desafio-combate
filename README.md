## 📌 Sobre o Projeto

Simulador de combate desenvolvido em **Java**, utilizando conceitos de **Programação Orientada a Objetos (POO)**.

O programa cria dois campeões e executa um combate por turnos, onde ambos se atacam simultaneamente até que um deles morra ou o número máximo de turnos seja atingido.

---

## 🧠 Funcionamento

- Cada campeão possui: **nome, vida, ataque e armadura**
- O dano é calculado com base no ataque do oponente descontando a armadura
- O campeão sempre perde **no mínimo 1 ponto de vida**
- A vida nunca fica negativa
- O combate termina quando um campeão chega a 0 de vida

---
